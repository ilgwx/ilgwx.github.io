# 个人主页

这是我的GitHub Pages个人主页，使用HTML和CSS构建。

## 功能

- 个人信息展示
- 项目展示
- 技能展示
- 响应式设计

## 如何部署

1. 确保你已经登录GitHub CLI：`gh auth login`
2. 初始化git仓库：`git init`
3. 添加文件：`git add .`
4. 提交更改：`git commit -m "Initial commit"`
5. 创建GitHub仓库：`gh repo create ilgwx.github.io --public --source=. --remote=origin`
6. 推送到GitHub：`git push -u origin main`
