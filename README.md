# CampusCycle

CampusCycle 是一个专为大学生设计的循环经济平台，让学生可以轻松回收、转售和管理毕业季的闲置物品。

## 核心功能

- **个人出售**：用户自主上架物品，自己定价，管理售后
- **官方出售**：直接出售给平台，获得即时现金
- **AI价格建议**：基于市场数据的智能定价系统
- **校园匹配**：同一大学内的买家和卖家匹配
- **校园取货**：校内物流服务
- **一键宿舍清理**：毕业生专属服务

## 技术特点

- 响应式设计，适配不同设备
- 平滑的水平滚动导航
- 现代化的UI设计，类似Apple/Airbnb风格
- 数据统计和环境影响展示
- 学生用户评价
- 强行动号召按钮

## 项目愿景

CampusCycle 旨在通过循环经济理念，减少校园浪费，为学生创造额外收入，同时为环保事业做出贡献。
